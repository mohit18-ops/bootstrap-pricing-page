# bootstrap-pricing-page
#Description
This project is a simple and responsive pricing page built using HTML5 and Bootstrap 5. It showcases a clean subscription pricing layout with three different plans: Free, Pro, and Enterprise. Each plan is presented using Bootstrap cards for a structured and visually appealing design.
